# Triple Espresso

Este projeto foi desenvolvido como parte do curso de Desenvolvimento Web da TripleTen.

## Descrição

O site apresenta uma cafeteria fictícia chamada Triple Espresso. Ele contém:

- Cabeçalho com navegação;
- Seção de receitas com vídeos incorporados;
- Formulário para reserva de mesa;
- Rodapé com informações de contato e redes sociais.

## Tecnologias utilizadas

- HTML5
- CSS3
- Metodologia BEM
- Flexbox
- Normalize.css
